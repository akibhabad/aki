# Aki Personal Website - Project Context

## Core idea
This site is a digital thinking archive, not a resume site and not a generic student portfolio.

It should feel like an online home for:
- writing
- ideas
- current curiosities
- curated projects
- books and reflections
- personal intellectual identity

## Public site structure
- Home
- Writing
- Projects
- Curiosities
- Library
- About
- Contact

## Homepage order
1. Hero
2. Short identity / thesis
3. Featured writing
4. Current curiosities
5. Selected projects
6. Library preview
7. About preview
8. Contact / footer

## Most important conceptual sections
1. Writing
2. Curiosities

These should feel like the center of gravity of the site.

## Brand traits
- curious
- intellectually serious
- ambitious
- technical
- entrepreneurial
- aesthetically strong
- thoughtful
- memorable

## Visual direction
- dark cosmic / near-black backgrounds
- subtle galaxy / starfield / observatory feel
- violet / purple glow accents
- serif display headings + sans body text
- restrained motion
- premium and editorial

## Anti-goals
- no generic portfolio feel
- no resume-site feel
- no crypto aesthetic
- no gamer aesthetic
- no cheesy sci-fi
- no over-animation
- no clutter
- no generic AI-branding copy
